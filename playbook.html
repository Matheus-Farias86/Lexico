O Playbook Prático

Guia de Implementação: Do Conceito ao Código

1. Jan Tschichold & A Nova Tipografia

Conceito: Hierarquia visual e clareza técnica. Tschichold nos ensinou que a tipografia deve ser funcional e organizada por escalas lógicas.

Título H1 (Escala Fluida)

Este é um parágrafo usando a fonte serifada para leitura editorial, garantindo contraste e legibilidade conforme os princípios de Tschichold.

/* Design Tokens de Tipografia */
--font-sans: 'Google Sans', sans-serif;
--font-serif: "Georgia", serif;
--fs-h1: clamp(1.75rem, 4vw, 2.5rem);
--leading-normal: 1.5;

2. De Stijl & Primitivas de Cor

Conceito: Redução ao essencial. As cores no Léxico são tratadas como Design Tokens puros, agindo como as cores primárias de Mondrian para sustentar a estrutura visual.

Primary Token

#5D4037 - Usada para ações e destaque estrutural.

Surface Token

#FAF9F6 - A base neutra que permite o respiro visual.

/* Cores como Primitivas Históricas */
--color-action-primary: #5D4037;
--color-bg-secondary: #FAF9F6;

3. Swiss Design & Sistemas de Grids

Conceition: O rigor da grade modular. Utilizamos CSS Grid para replicar o sistema proporcional suíço, garantindo alinhamento e uso consciente do espaço negativo.

Módulo 02

/* Implementação do Grid Suíço */
.cards-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: var(--space-md);
}

4. Bauhaus & Componentes Funcionais

Conceito: Forma segue a Função. Nossos componentes (Botões e Cards) são peças industriais reproduzíveis, focadas inteiramente na utilidade técnica.

Primário Outline
/* Componente BEM (Bauhaus Style) */
.btn {
  padding: var(--space-sm) var(--space-md);
  text-transform: uppercase;
  font-weight: var(--fw-bold);
}
